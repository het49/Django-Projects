# BeerBuddy (Web App)

This folder contains the Django project for the BeerBuddy web application.

## Where users go

- Platform/Admin portal:
  - Sign in: `/web/users/login/`
  - Dashboard: `/web/users/dashboard/`
  - Brewery registration & management: `/web/users/brewery_registration/`, `/web/users/brewery_management/`
- Brewery portal:
  - Sign in: `/web/brewery/login/`
  - Brewery profile: `/web/brewery/profile/`
  - Offers: `/web/brewery/offers/`

If you don’t see a screen you need, it’s usually because your account doesn’t have permission for it.
